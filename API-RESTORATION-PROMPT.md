# Restoration Prompt

This prompt removes all artifacts generated by the implementation prompt and restores the package to its pre-implementation state.

**Warning**: This operation deletes source code, tests, and generated configuration. However, all these files can be recreated by running the API-IMPLEMENTATION-PROMPT.md again. The specification files are preserved.

---

## Part 1: Verify Package State

Before proceeding with restoration, verify that the package is in a valid state:

1. **Verify specification files exist**:
   - [ ] `/docs/api/specs/hello-endpoint.md` exists
   - [ ] `/docs/api/specs/api-design.md` exists
   - [ ] `/docs/api/specs/error-handling.md` exists
   - [ ] `/docs/api/specs/technology-choices.md` exists
   - [ ] `/docs/api/specs/getting-started-design.md` exists

2. **Verify core guidance files exist**:
   - [ ] `CLAUDE.md` exists
   - [ ] `README.md` exists
   - [ ] `API-IMPLEMENTATION-PROMPT.md` exists

If any specification or core file is missing, **STOP** and verify the package integrity before proceeding.

## Part 3: Delete Generated Artifacts

### Step 1: Remove Source and Test Code

```bash
# Remove source code directory
rm -rf src/api/

# Remove tests directory
rm -rf tests/api/

# Verify deletion
echo "Verifying deletion of source code..."
[ ! -d src/api ] && echo "✓ src/ removed" || echo "✗ src/ still exists"
[ ! -d tests/api ] && echo "✓ tests/ removed" || echo "✗ tests/ still exists"
```

### Step 2: Remove Configuration Files

```bash
# Remove SDK version lock
rm -f global.json

# Remove solution file
rm -f hello-world-api.sln

# Remove .gitignore
rm -f .gitignore

# Verify deletion
echo "Verifying deletion of configuration files..."
[ ! -f global.json ] && echo "✓ global.json removed" || echo "✗ global.json still exists"
[ ! -f hello-world-api.sln ] && echo "✓ hello-world-api.sln removed" || echo "✗ hello-world-api.sln still exists"
[ ! -f .gitignore ] && echo "✓ .gitignore removed" || echo "✗ .gitignore still exists"
```

### Step 3: Remove Generated Documentation

```bash
# Remove generated getting-started guide
rm -f docs/api/guides/getting-started.md

# Verify deletion
echo "Verifying deletion of generated documentation..."
[ ! -f docs/api/guides/getting-started.md ] && echo "✓ docs/api/guides/getting-started.md removed" || echo "✗ docs/api/guides/getting-started.md still exists"
```

### Step 4: Clean Build Artifacts (Optional)

These are safe to delete as they're automatically generated by `dotnet build`:

```bash
# Remove build output directories
rm -rf src/api/api/bin/ src/api/obj/
rm -rf tests/api/api/bin/ tests/api/obj/

echo "Verifying deletion of build artifacts..."
[ ! -d src/bin ] && echo "✓ src/api/bin/ removed" || echo "✗ src/api/bin/ still exists"

[ ! -d tests/obj ] && echo "✓ tests/api/obj/ removed" || echo "✗ tests/api/obj/ still exists"
```

## Part 4: Verify Restoration

### Verify Specifications Remain

```bash
echo "=== Verifying specification files remain intact ==="

check_file() {
  if [ -f "$1" ]; then
    echo "✓ $1"
    return 0
  else
    echo "✗ $1 (MISSING - RESTORATION INCOMPLETE)"
    return 1
  fi
}

all_ok=true

check_file "CLAUDE.md" || all_ok=false
check_file "README.md" || all_ok=false
check_file "API-IMPLEMENTATION-PROMPT.md" || all_ok=false
check_file "docs/api/specs/hello-endpoint.md" || all_ok=false
check_file "docs/api/specs/api-design.md" || all_ok=false
check_file "docs/api/specs/error-handling.md" || all_ok=false
check_file "docs/api/specs/technology-choices.md" || all_ok=false
check_file "docs/api/specs/getting-started-design.md" || all_ok=false
check_file "docs/api/specs/restore-prompt-design.md" || all_ok=false
check_file "docs/api/guides/adding-endpoints.md" || all_ok=false
check_file "docs/api/adr/adr-001-minimal-apis.md" || all_ok=false

if [ "$all_ok" = true ]; then
  echo ""
  echo "✓ All specification files present"
else
  echo ""
  echo "✗ Some specification files are missing - restoration may be incomplete"
fi
```

### Verify Generated Files Are Gone

```bash
echo "=== Verifying generated files have been removed ==="

check_removed() {
  if [ ! -e "$1" ]; then
    echo "✓ $1 removed"
    return 0
  else
    echo "✗ $1 still exists"
    return 1
  fi
}

all_removed=true

check_removed "src/api" || all_removed=false
check_removed "tests/api" || all_removed=false
check_removed "global.json" || all_removed=false
check_removed "hello-world-api.sln" || all_removed=false
check_removed ".gitignore" || all_removed=false
check_removed "docs/api/guides/getting-started.md" || all_removed=false

if [ "$all_removed" = true ]; then
  echo ""
  echo "✓ All generated files removed"
else
  echo ""
  echo "✗ Some generated files remain"
fi
```
